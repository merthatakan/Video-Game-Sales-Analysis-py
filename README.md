# Video-Game-Sales-Analysis-py
I utilized Python to automate the data acquisition process from Kaggle and generated high-quality visualizations to understand the gaming industry's evolution. The project itself  is designed to demonstrate data cleaning, exploration (EDA), and visualization skills.


# Key Features
• **Automated Data Fetching**: Uses kagglehub API to pull the latest dataset.

• **Data Cleaning**: Handles missing values and ensures data types are correct for analysis.

•**European Market Focus**: Detailed analysis of the most popular genres and top publishers in Europe.

•**Global Trends**: Visualizes the rise and fall of game sales over the decades.

 # Tech Stack
• **Language**: Python

•**Libraries**: Pandas, Matplotlib, Seaborn, Kagglehub, OS

•**API**: Kaggle API

# Key Insights from the Data
•**Genre Dominance**: Action and Sports games historically dominate the European market.

•**The Golden Era**: Global video game sales reached their peak around 2008-2009, driven by the PS3, Xbox 360, and Wii era.

•**Market Leaders**: Companies like Nintendo and Electronic Arts (EA) show consistent leadership in European sales.
